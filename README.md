# MORPHY COOK OFFICIAL

Morphy Cook Official adalah aplikasi web resep makanan yang saya bangun sebagai project portfolio fullstack untuk mempelajari pengembangan aplikasi modern menggunakan ReactJS, ExpressJS, dan MySQL.

Melalui project ini saya mempelajari banyak hal, mulai dari autentikasi pengguna, role-based access control, integrasi REST API, manajemen state, pengelolaan database relasional, hingga implementasi Two-Factor Authentication (2FA).

Project ini masih terus dikembangkan dan akan mendapatkan peningkatan pada sisi UI/UX, keamanan aplikasi, performa, serta penambahan fitur-fitur baru di masa mendatang.
